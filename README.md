# Linear-LexBFS
This is a linear implementation of LexBFS algorithm

# Input file Sample:
18725	197084

0	1

0	2

0	3

0	4

1	14

1	16

1	159

1	160

1	161

2	148

...

The first line specifies the number of nodes and the number of edges respectively. Each of next lines specifies a connection between 2 nodes.
